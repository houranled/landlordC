# landlordC
实现斗地主核心算法逻辑。可通过控制台字符界面进行操作。

![image](https://github.com/user-attachments/assets/ad273aa0-01ae-4e7d-9c45-fbd3399dfc95)

## Compile
  gcc -o landlord main.c op.c 

## Play
  ./landlord
